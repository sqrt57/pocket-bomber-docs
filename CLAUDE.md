# Pocket Bomber — Docs repo

Design documents for the Pocket Bomber game. Source code is in the `source/` repo.

## Structure

- `gdd.md` — full game design document (single source of truth for rules, mechanics, scope)
- `architecture.md` — component hierarchy, game loop flow, key files

## Policy

Always on `main`. No branches. Fast-forward only (`merge.ff = only`).
